# RoflCamera 📱📷
Превращает ваш iPhone в высококачественную веб-камеру для ПК по проводу (USB) или Wi-Fi, с оптимизацией и энергосбережением!

## Как получить приложение на iPhone (IPA)
Поскольку у вас нет Mac, вы можете легко скомпилировать приложение через GitHub Actions без участия разработчика:

1. Загрузите файлы из этой папки в ваш репозиторий на GitHub.
2. В GitHub перейдите на вкладку **Actions**.
3. Выберите `Build iOS IPA` слева и нажмите **Run workflow**. (Или он запустится сам при коммите).
4. После завершения сборки скачайте артефакт `RoflCamera.ipa` внизу страницы сборки.
5. Установите `.ipa` файл на ваш iPhone используя **Sideloadly**.

## Приложение для ПК (Windows)
Программа для включения веб-камеры на Windows также готова и лежит в папке `RoflCamera-PC`. Она написана на Python.

### Установка (Windows):
1. Установите Python 3 (желательно с официального сайта python.org, при установке поставьте галочку "Add Python to PATH").
2. Откройте командную строку (`cmd`) или PowerShell в папке `RoflCamera-PC`.
3. Установите зависимости:
   ```cmd
   pip install -r requirements.txt
   ```
4. Запустите саму программу:
   ```cmd
   python main.py
   ```

### Как использовать:
- **По проводу (USB):** Выберите "USB (Кабель)" в программе на ПК и нажмите "Запустить". Программа *автоматически* пробросит порт с телефона на компьютер (через встроенную библиотеку `pymobiledevice3`, даже без установленного iTunes!).
- **По Wi-Fi:** Выберите Wi-Fi и введите IP, который показывает приложение на айфоне.
- **В OBS Studio:**
  1. Вы можете использовать "Источник медиа" и вставить ссылку `http://127.0.0.1:8080`, если подключаетесь по USB.
  2. Либо, чтобы камера была доступна во всем Windows (Discord, Zoom), установите **OBS Virtual Camera** (встроена в новые версии OBS) — наша программа сама автоматически начнет передавать кадры в систему под видом веб-камеры!
  
### Фишки для аккумулятора:
В приложении на iPhone есть кнопка **"Выключить экран (Энергосбережение)"**. При её нажатии:
- Яркость экрана гасится до 0 (на OLED экранах iPhone 13 Pro это означает, что каждый черный пиксель физически отключается и вообще не потребляет энергию и не греется).
- Автоблокировка устройства (которая ставит систему в сон) отключается, поэтому поток идет без обрывов.
- Приложение само выключает экран если 30 секунд не было касаний при активной съемке.
- Чтобы "разбудить" телефон, достаточно просто коснуться черного экрана.
